  1. 01

    多智能体自动化标签挖掘系统（AgentFlow）

    主导建设 · 2026.01 – 至今

    针对银行征信数据缺失，利用微信行为数据构建白名单用户婚姻状况预测模型。预研发现离婚群体违约风险突出，逾期 30+ Lift 高达 1.81。

    传统建模：设计两阶段级联建模方案，在 XGBoost 基础上探索 TabPFN 模型，OOT 样本 AUC 0.75 / KS 0.37；成功补齐 9957 万用户婚姻标签，模型评分 Top 3% 区域风险识别率达大盘 1.51 倍。

    Agent 探索：构建 AgentClassifyPipeline 通用框架，打通"特征提取 → SQL 生成 → 逻辑判定 → 报告生成"全流程闭环，代码已合入微信支付代码库。引入 RAG + Few-shot + CoT + MoE 并行专家矩阵，并通过 logprob 机制量化分类置信度。

    项目成果：捕获 XGBoost 漏召回的 10% 难识别样本；维持 60% 召回率前提下，查准率较 XGBoost 提升 20%（达 43%）。相关成果已发表腾讯内网 KM 技术好文，并申请发明专利《一种基于大语言模型的自动化标签挖掘方法及系统》。

  2. 02

    Agent 自动化优化框架（分付规则解释助手）

    独立负责

    针对分付规则答疑 Agent 迭代依赖人工试错、归因不可追溯、改 Prompt/KB 易引入震荡等痛点，引入 TextGrad 文本梯度范式，将"反向传播 + 梯度下降"迁移到 LLM 系统优化，自研"评测 + 自动归因 + 优化 + 回归"全流程闭环 Skill。

    框架设计：将 Agent 类比神经网络计算图——Prompt / KB 为文本变量，Adams 答题为 Forward，Gemini 评分为 Loss，CodeBuddy 切换"归因 / 优化"双角色实现 Backward + Optimizer。

    核心设计：四象限结构化归因（每条改动可溯源至源 case）；Backward / Optimizer 角色分离 + 宪法红线；Momentum 历史注入 + 单轮单主题聚焦；Human-in-the-loop 四道闸门兼顾自动化与业务安全。

    项目成果：261 道基线 case 经四轮调优，0 分率从 25% 降至 3%，溯源率 100%，单轮迭代耗时从 2 h 降至 20 min，已沉淀为可复用 Skill 工具。

    TextGradLLM AgentHarness 评测 Prompt 优化自动化迭代

  3. 03

    基于因果推断的贷中降额策略与连续干预建模

    深度参与

    针对降额管控后违约率不降反升 3.64% 的"降额摆烂"现象，进行因果效应建模，识别降额后的正向作用人群。

    离散干预建模：实验数据采用 Meta-Learner 框架 + Bagging 降低预估方差，逾期金额归一化重加权对齐业务目标；观测数据采用 DragonNet，倾向性预估实现特征蒸馏消除混杂变量；推导局部干预对整体收益判别式，根据 Qini 指数将决策点定在 31 分位点。

    连续干预探索：引入 VCNet 解决最优降额幅度决策难题，样条基函数将网络参数建模为干预强度 t 的动态函数，拟合个体化"剂量-响应曲线"。

    项目结果：DragonNet 方案使管控客群人均在贷金额下降 3.46%，人均逾期金额下降 13.35%，整体客群金额违约率下降 0.38%。

  4. 04

    大模型驱动的发票用户标签体系构建与业务应用

    主要负责人 · 2026.04

    针对 Agent 挖掘出的开放式发票标签无法直接进入业务圈选的问题，建设从自由标签生产到稳定业务标签体系的归纳与治理链路。

    问题诊断：AgentFlow 产出 2,338 种自由标签，其中约 76% 为单点长尾；标签命名、粒度和归属不统一，产品难以用其直接判断可运营人群。

    体系建设：按“分类 → 去重 → 合并 → 归纳 → 简化”处理开放标签，为每类沉淀 Summary 与 evidence；标签库作为圈选、效果评估和敏感标签治理的统一输入，并通过产品评审持续回流修正。

    项目成果：把一次性生成的标签结果转化为可入库、可圈选、可持续迭代的标签资产；AgentFlow 作为上游能力保留丰富语义和解释证据。

  5. 05

    信贷用户生命周期价值 pLTV 预测与支付长序列表征探索

    参与实验探索 · 贷前 / 未开通 / 历史有开通客群

    围绕 pLTV 同时受用户响应和风险成本影响的双任务问题，探索宽表 Embedding、专家网络和支付行为长序列表征的组合方式。

    表征与基线：将宽表文本化后注入 KaLM 896 维 Embedding；实验发现其分类增益较弱、回归增益更明显，因此采用“原始特征做分类 + Embedding 做回归”的两阶段 LGBM 组合。

    模型探索：通过 PLE / CGC 区分共享与任务独占专家，缓解响应与风控负迁移；使用 ZILN 表达零膨胀价值分布，并尝试以 FiLM / score 将支付长序列表征注入任务塔。

    实验记录：相对基线，全量与支用误差分别下降 31.8%、28.3%；同时记录 OOT 漂移低估问题，作为后续稳定性优化方向。

Spec-PINN：算子诱导流形嵌入的物理信息神经网络

Physical Information Neural Network Based on Operator-Induced Manifold Embedding for Wave Propagation and Medium Inversion

Engineering Applications of Artificial Intelligence (EAAI) · 2025 · 第一作者

针对物理信息神经网络（PINN）在求解高频 Helmholtz 方程时的两大顽疾—— 谱偏差（高频振荡难拟合）与逆问题"周期跳跃"（非凸优化陷局部最优）， 提出 Spec-PINN 框架，把解析物理性质内化进网络结构：

  • 正问题：构造由基本解算子诱导的物理核嵌入层，从根源消除数值色散；
  • 逆问题：设计渐近多尺度基双网络架构，将棘手的频率搜索转化为鲁棒的权重优化；
  • 非均匀介质 benchmark 上，相比 FE-HRKAN 等 SOTA 方法在精度与收敛速度上均有显著提升。

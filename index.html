  1. 01

    多智能体自动化标签挖掘系统（AgentFlow）

    主导建设 · 2026.01 – 至今

    传统建模：针对银行征信婚姻字段缺失，设计两阶段级联方案，在 XGBoost 基础上探索 TabPFN，OOT 达 AUC 0.75 / KS 0.37；补齐 9957 万用户婚姻标签，离婚标签 Top 3% 风险识别率为大盘 1.51 倍。

    Agent 探索：针对样本依赖重、新场景难复用的问题，搭建 LLM 自动化标签挖掘链路，覆盖特征提取、SQL 生成、逻辑判断和报告生成；引入结构化数据语义化、RAG Few-shot、意图路由、CoT、并行专家和 logprob 置信度量化。

    微调与蒸馏：外部版进一步展示专用微调和小模型蒸馏探索：用 Qwen3.5-9B + LoRA/DoRA 做教师，再蒸馏至 Qwen3.5-0.8B + LoRA，学生 OOT 达 AUC 0.71 / KS 0.41，接近教师效果。

    项目成果：Agent 侧补回 XGBoost 漏召约 10% 难例，60% 召回下查准率相对 XGBoost 提升 20%（达 43%）；成果沉淀为腾讯 KM 技术好文，并申请发明专利。

  2. 02

    Agent 自动化优化框架（分付规则解释助手）

    独立负责

    项目背景：针对分付规则答疑 Agent 依赖人工试错、归因不可追溯、改 Prompt / KB 易震荡的问题，引入 TextGrad 文本梯度思想，自研评测、归因、优化和回归闭环。

    Harness 建设：搭建 261 题固定基线评测流水线，批量调用 AgentFlow 答题，由统一 Judge 打分，并强制采集 query 改写、召回详情等 RAG 中间态，支持跨轮 diff 和 case 级归因。

    框架设计：将 Prompt / KB 视为文本变量，答题为 Forward，Judge 为 Loss；设计 Prompt vs KB 五类归因决策树、Momentum 防震荡、mini-batch 单主题迭代、Validation Gate 和四道人机闸门。

    项目成果：四轮调优后 0 分率由手工阶段约 25% 稳态降至约 3.8%，改动溯源率 100%，单轮迭代由人工约 2h 降至约 20min，成果沉淀为 KM 技术分享与可交接 Skill。

    TextGradLLM AgentHarness 评测 Prompt 优化自动化迭代

  3. 03

    基于因果推断的贷中降额策略与连续干预建模

    深度参与

    项目背景：针对降额管控后违约率不降反升 3.64% 的“降额摆烂”现象，进行因果效应建模，识别真正适合降额的正向作用人群。

    离散干预建模：实验数据采用 Meta-Learner + Bagging 降低预估方差，并用逾期金额归一化重加权对齐业务目标；观测数据采用 DragonNet，通过倾向性预估任务做特征蒸馏，缓解混杂变量影响。

    连续干预探索：针对“降多少”的问题引入 VCNet，用样条基函数将网络参数建模为干预强度 t 的动态函数，拟合个体化剂量-响应曲线，为差异化定额提供依据。

    项目结果：测试集上 DragonNet 方案使管控客群人均在贷金额降低 3.46%，人均逾期金额降低 13.35%，整体客群金额违约率下降 0.38%。

  4. 04

    大模型驱动的发票用户标签体系构建与业务应用

    主要负责人 · 2026.04

    项目背景：发票数据信息密度高，但长期难转化为稳定用户画像；传统标签建设人力重、扩展慢，因此探索用 AgentFlow 快速挖掘可解释标签，再归纳为可持续消费的发票标签体系。

    工作流设计：搭建 AgentFlow，对陌生或长尾公司按需检索补知识，产出带 evidence 的涌现标签与自然语言 Summary；评测集产出约 2338 种标签，大厂员工金标相对旧模型 Accuracy +10.2pp、Precision +18.7pp。

    工程提效：针对约 171 万开通有发票用户，用 prefix cache、搜一搜截断和模型选型，将全量生产由约 10 天压至 2–6 天。

    项目成果：针对约 76% 长尾涌现，建立词频统计、映射规则、L1-L3 taxonomy、用户标签长表 / 宽表的归纳链路；进一步用于 HVLR 人群识别，将高净值低风险客群口径落到可解释标签组合，并用风险 Lift 验证业务圈选效果。

  5. 05

    信贷用户生命周期价值 pLTV 预测与支付长序列表征探索

    参与实验探索 · 贷前 / 未开通 / 历史有开通客群

    项目背景：信贷 pLTV 需要同时刻画借款意愿与违约风险，两类目标在特征空间存在对抗；金额侧还有零膨胀与长尾问题，传统分模型再相乘容易误差叠加。

    KaLM Embedding：将宽表字段写成结构化文本，输入微信通用文本向量模型得到 896 维表征；对照纯 Embedding、特征截断、拼接宽表和 PLS 等方案后，采用“原始特征分类 + KaLM Embedding 回归”的混合两阶段。

    PZ-Net 方法对齐：围绕 PLE / CGC 多专家门控、ZILN 零膨胀金额分布和 Sigma Penalty，处理响应与风控负迁移；响应 AUC 达 0.7982，pLTV 总体误差 11.49%，较 MMoE+ZILN 的 15.38% 相对降低约 25%。

    实验记录：全量/支用样本误差分别下降 31.8%/28.3%，Top10% 金额误差约降 20.5%；支付长序列表征融合后，响应 AUC 0.7982 → 0.8108，Top10% 响应 39.85% → 41.16%。

Spec-PINN：算子诱导流形嵌入的物理信息神经网络

Physical Information Neural Network Based on Operator-Induced Manifold Embedding for Wave Propagation and Medium Inversion

Engineering Applications of Artificial Intelligence (EAAI) · 小修阶段 · 第一作者

针对物理信息神经网络（PINN）在求解高频 Helmholtz 方程时的两大顽疾—— 谱偏差（高频振荡难拟合）与逆问题"周期跳跃"（非凸优化陷局部最优）， 提出 Spec-PINN 框架，把解析物理性质内化进网络结构：

  • 正问题：构造由基本解算子诱导的物理核嵌入层，从根源消除数值色散；
  • 逆问题：设计渐近多尺度基双网络架构，将棘手的频率搜索转化为鲁棒的权重优化；
  • 非均匀介质 benchmark 上，相比 FE-HRKAN 等 SOTA 方法在精度与收敛速度上均有显著提升。
